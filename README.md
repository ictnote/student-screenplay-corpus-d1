# Student Screenplay Corpus for D1 Assessment

This repository contains twelve anonymized student screenplays used in the study:

“Exploratory Evaluation of Large Language Models for Three-Act Structure and Pacing Assessment in Student Screenplays.”

## Corpus

The files `N01.pdf` to `N12.pdf` contain anonymized student screenplays evaluated using the D1 rubric dimension: three-act structure and story pacing.

## Qualitative Cases Discussed in the Paper

The cases discussed in Table XI are:

- Highest-rated screenplay: `N12.pdf`
  - Teacher 1 = 100
  - Teacher 2 = 100
  - Teacher reference = 100

- Lowest-rated screenplay: `N08.pdf`
  - Teacher 1 = 60
  - Teacher 2 = 40
  - Teacher reference = 50

## Purpose

This corpus is provided to support the qualitative illustration of three-act structure and pacing discussed in the associated paper.

## Data Protection Notice

All direct identifiers and identifiable metadata have been removed or replaced with anonymized screenplay codes.

Users must not attempt to identify, contact, profile, or re-identify the student authors.

## Scope

This repository contains screenplay files only. Teacher scores, LLM outputs, prompt configurations, and experimental analysis files are not included.
